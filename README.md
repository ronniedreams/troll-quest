# TROLL QUEST

A platformer that hates you. 50 levels, ~50 distinct troll mechanics, a boss fight, and an upside-down finale.

**Play:** open `index.html`, or visit the GitHub Pages URL for this repo.

- Keyboard: arrows / WASD + Space · P pause · L level select · M mute · R restart
- Touch: hold left half of screen to steer, right half to jump
- Gamepad: full support (A jump · X levels · Y mute · Start pause · Select restart)

Progress auto-saves in your browser (localStorage). Built as a single self-contained HTML file — no dependencies, no build step.
